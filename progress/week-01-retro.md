- Day 1:  Pertama kali buka terminal. Belajar navigasi folder, bikin file, dan install VS Code.

Disinilah saya belajar yang namanya version control system, dimana kita bisa mengabadikan kode kita di github dan juga bisa mengembalikannya kapanpun, dan semua developer profesional pasti pakai ini buat bekerja tim.
