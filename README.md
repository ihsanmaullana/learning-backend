# Learning Backend

A 365-day journey from zero to backend engineer — documented one day at a time.

---

## About

This repository tracks my daily learning progress as I build from the ground up toward
becoming a professional backend engineer. Every day has notes, exercises, and reflections
so I can look back and see how far I've come.

Part of a larger **1000-day fullstack engineering journey** — starting from absolute zero,
heading toward fullstack engineer, accessibility specialist, and startup founder.
