# Day 1 — Terminal, File System & VS Code Setup

## Key Concepts

- **Terminal (CLI)**: A text-based interface to communicate directly with the computer
- **GUI vs CLI**: GUI uses visual elements (buttons, menus), CLI uses text commands
- **File System**: How files and folders are organized on a computer

## Commands Learned

| Command | What it does |
|---------|-------------|
| `pwd` | Print working directory — shows where I am |
| `ls` | List files and folders |
| `cd` | Change directory — navigate between folders |
| `mkdir` | Make a new directory (folder) |
| `touch` | Create an empty file |
| `cat` | Display file contents |
| `clear` | Clear terminal screen |
