# Day 17 — Object Methods

## Apa itu Object Methods?
Metode untuk memanipulasi object terutama ketika untuk diambil atau di tampilkan

## Object.keys()
Digunakan untuk mengambil data property nya saja dari suatu object

## Object.values()
Untuk mengambil value atau nilainya saja dari suatu object dan property

## Object.entries()
Digunakan untuk menghasilkan keys dan value 

## Hubungan dengan yang Sudah Dipelajari
Sebelumnya belajar terkait filter, map, dan for each, tentu ini bisa digunakan bersamaan dengan ini data bisa di manipulasi, di filterasi atau ditampilkan dengan yang kita mau dengan format terbaik dan ter aman

## Poin Penting yang Tidak Boleh Dilupakan
pastikan untuk menampilkan data object tidak langsung semua ditampilkan ke front end karna itu berbahaya buat security takutnya banyak data sensitif yang ditampilkan yang seharusnya di rahasiakan

## Security Notes
Jangan tampilkan data object sebelum di filterasi
